# World History Study Hub

A self-contained, offline-friendly study app for an accelerated summer-school **World History** course (Semester 1: exploration → revolutions → industrialization → imperialism → an age of democracy & progress).

**Open it:** https://yvesbrancheau.github.io/beau-world-history-study/

Includes:
- 🃏 Flashcards with spaced repetition
- ✅ Multiple-choice quizzes by topic + a 66-question mock final
- ✍️ Essay trainer (Claim–Data–Warrant format) with model essays
- 📚 Searchable reference of key terms, people, and events

Everything runs in the browser. Progress (flashcard schedule, quiz stats, saved drafts) is stored locally on your device — nothing is sent anywhere.

> A study aid for learning and practice only.
